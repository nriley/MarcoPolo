Setting up MarcoPolo's actions

Actions are triggered when MarcoPolo decides to change your context.

Each action has the following properties:

  • A type,
  • A parameter for the type of action,
  • A set of triggers that will cause the action to run (see below),
  • A delay between changing context and triggering the action,
  • Whether it is enabled, and
  • A description for human readability.

Types of triggers are:

  • Arrival at a context,
  • Departure from a context,
  • Computer's Wake from sleep,
  • Computer's Sleep, and
  • MarcoPolo's Startup.

In most cases, the delay parameter will be the length of time from the trigger to when the action is executed. Departure triggers work in reverse: the delay parameter is the length of time preceeding the context change when the action will be run.

Adium actions

Actions of type Adium can change the IM status if you use Adium (requires version 1.2 or later).

DefaultPrinter actions

Actions of type DefaultPrinter change your computer's default printer.

DesktopBackground actions

Actions of type DesktopBackground will change your desktop background picture.

FirewallRule actions

Actions of type FirewallRule can enable or disable the network firewall for various services. This action is currently not functional under Mac OS X 10.5 (Leopard).

iChat actions

Actions of type iChat change your iChat status message.

MailIMAPServer actions

Actions of type MailIMAPServer change the default IMAP server of your Mail.app accounts.

MailSMTPServer actions

Actions of type MailSMTPServer change the default SMTP server of your Mail.app accounts.

Mount actions

Actions of type Mount attempt to mount a network volume (afp://, smb://, etc.).

Mute actions

Actions of type Mute will mute or unmute your system audio.

NetworkLocation actions

Actions of type NetworkLocation will change your current Mac OS X Network Location.

Open actions

Actions of type Open are the most general actions. Upon triggering, MarcoPolo will open that file: if it is an application, it will be launched; if it is an AppleScript, it will be executed; otherwise, it will be opened with that file's default application. This is roughly equivalent to double-clicking on that file in Finder.

OutputVolume actions

Actions of type OutputVolume can change the volume level of your system audio output.

PowerManager actions

Actions of type PowerManager allow configuration of your computer's power management settings.

QuitApplication actions

Actions of type QuitApplication cause a named application to quit.

ScreenSaverPassword actions

Actions of type ScreenSaverPassword either enable or disable your screen saver's password.

ScreenSaverStart actions

Actions of type ScreenSaverStart either start or stop your screen saver.

ScreenSaverTime actions

Actions of type ScreenSaverTime can change the idle time before your screen saver activates.

ShellScript actions

Actions of type ShellScript allow you to execute arbitrary shell scripts using /bin/sh. Arguments may be passed to the script by using a pipe character ("|") as a separator in the parameter. (e.g. /path/to/script.sh|argument1|argument2)

SleepTime actions

Actions of type SleepTime can modify the idle time before sleeping some things (computer, disk and display).

TimeZone actions

Actions of type TimeZone will change your computer's time zone.

ToggleBluetooth and ToggleWiFi actions

Actions of type ToggleBluetooth or ToggleWiFi allow you to control the power state of your computer's built-in Bluetooth and WiFi controllers.

Unmount actions

Actions of type Unmount attempt to unmount a named volume.

VPN actions

Actions of type VPN can establish or disconnect from your default PPTP or L2TP VPN.
