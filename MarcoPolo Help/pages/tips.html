MarcoPolo Help

Tips and Tricks

This page gives examples of useful ways that MarcoPolo can be used, especially via actions.

Where an AppleScript is given, it should be created and saved with the Script Editor, and pointed to by an Open action in MarcoPolo.

To cause Adium to connect and set your status:

Use the following AppleScript: 

tell application "Adium"
	activate
	set my status message to "At home."
	set my status type to available
end tell

You may replace available with away to have Adium set your status as busy.

To launch iTunes and hide its window:

Use the following AppleScript: 

tell application "iTunes"
	launch
	set visible of window 1 to false
end tell

To unlock the keychain:

Use the following AppleScript: 

tell application "Keychain Scripting"
	unlock
end tell

You may replace unlock with lock to reverse the operation.
